# Platforma

Браузерна desktop-версія ISMS-платформи з графічним інтерфейсом, іконками та «робочим столом» для навігації між модулями.

## Що реалізовано
- Браузерний desktop-інтерфейс з боковим доком (іконки-модулі).
- Графічні картки для клауз 4–10, Додатку A та аудиту.
- Окремі вікна для контролів, ризиків і завдань.

## Запуск
Варіант 1 (найпростіше):
```bash
python3 -m http.server 8080
```
Потім відкрийте [http://localhost:8080](http://localhost:8080).

Варіант 2:
просто відкрийте `index.html` у браузері.
