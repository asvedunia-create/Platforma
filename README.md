# Platforma

Працюючий браузерний desktop для ISMS-платформи: іконки на робочому столі, вікна модулів, таскбар, годинник, перетягування вікон.

## Можливості
- Desktop-іконки для модулів: Огляд, Контролі, Ризики, Завдання.
- Вікна як у desktop-системі: відкрити, мінімізувати, закрити.
- Drag-and-drop переміщення вікон та фокус по z-index.
- Таскбар з кнопкою швидкого відкриття всіх модулів.

## Запуск
```bash
python3 -m http.server 4173
```
Після цього відкрийте `http://localhost:4173`.
